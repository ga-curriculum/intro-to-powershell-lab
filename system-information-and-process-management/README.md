</h1>

**Learning objective:** In this exercise, you'll learn how to retrieve system information and perform basic process management tasks using PowerShell.

PowerShell provides a wealth of cmdlets for gathering system information and managing processes. Let's explore some useful commands that will help you gain insights into your system and control running processes.

## Commands to Learn

- `Get-ComputerInfo`: Retrieves detailed information about the computer system.
- `Get-Process`: Displays the processes running on the local computer.
- `Stop-Process`: Stops a running process.
- `Get-Service`: Retrieves information about services on the local computer.
- `Start-Service`: Starts a service.
- `Stop-Service`: Stops a service.

## Practice

1. Use `Get-ComputerInfo` to retrieve information about your computer system, such as the operating system, memory, and processor.
2. Use `Get-Process` to display a list of running processes on your computer.
3. Identify a non-critical process (e.g., a text editor) and stop it using `Stop-Process`.
4. Use `Get-Service` to retrieve a list of services running on your computer.
5. Choose a non-essential service (e.g., a print spooler) and stop it using `Stop-Service`. 
6. Then, start it again using `Start-Service`.
